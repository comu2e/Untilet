# Untilet

## #流れ()

1. Yunからテザリング回線を通じてM2X APIに2sおきにデータがプッシュされる。

2. ボタン操作、ViewDidLoadされたときに次の処理に関するメイン関数LoadApi()を呼び出すようにする

*LoadApi()

3. 関数を呼び出したときの時刻を取得

4. APIにたまったJSON データをalamofireで通信処理(JsonファイルをJsonDataとする)

5. JSONデータをJSONパーサー unboxあるいはswifty jsonでパース処理(この値をvalue(int型とする））(12/2完了） 

6. 値を計算する
  - 関数名をcalc_percent()とする
  - 引数は4の値valを使う
  - 戻り値はpercent
  - 処理内容は percent=(val-400)/100とする
  - valの値によってpercentを条件分岐する関数おも入れておく
   - 800 < val < 1023の時100％のようにしておいてもいい
